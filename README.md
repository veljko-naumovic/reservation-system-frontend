Booking App

Simple booking and reservation management application built with Next.js 14 (App Router), TypeScript, Redux Toolkit, and Tailwind CSS.

This project demonstrates CRUD operations, filtering, sorting, client/server components separation, metadata handling, and clean UI architecture.

Features
Bookings List

View all bookings

Filter by status (Pending / Confirmed / Cancelled)

Sort by newest / oldest

URL-synced filters

Empty state

"No results for selected filters" state

Create Booking

Form validation

Date validation (end date must be after start date)

Redirect after successful creation

Edit Booking

Pre-filled form

Update existing booking

Redirect to details page

Booking Details

View complete booking information

Change status directly from details page

Delete booking

Edge state: "This booking was deleted"

Delete Flow

Confirmation

Toast notification after delete

Proper redirect handling
